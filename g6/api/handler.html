Handler

概述

管理器，主要用来容纳、管理 模式 mode，行为 behaviour。

方法

dragHideTexts {Function}

拖拽隐藏文本

wheelZoomHideTexts {Function}

缩放隐藏文本

dragEdgeEndHideAnchor {Function}

多拽边结束后隐藏目标节点的锚点

dragNodeEndHideAnchor {Function}

多拽节点结束后隐藏目标节点的锚点

hoverAnchorSetActived {Function}

鼠标悬锚点激活锚点

hoverNodeShowAnchor {Function}

鼠标悬浮节点显示锚点

buttonPointer {Function}

当鼠标移到 element.get('class') 含 button 的图形元素上，指针变成小手的行为。

clickActive {Function}

鼠标左键点击激活子项 item（激活行为互斥）的行为。

clickAddNode {Function}

点击增加节点的行为，适用于网图编辑器。

clickBlankClearActive {Function}

点击画布空白区域，清除激活子项的行为，适用于网图编辑器。

dragCanvas {Function}

拖动画布的行为。

dragBlank {Function}

拖动画布空白区域的行为。

dragAddEdge {Function}

拖动添加边的行为，适用于网图编辑器。

dragEdge {Function}

拖动边切换边连接的节点的行为。

dragNode {Function}

拖动节点行为。

multiSelect {Function}

多选行为。

resizeEdge {Function}

边变形行为。

resizeNode {Function}

节点变形行为。

wheelZoom {Function}

滚轮缩放行为。

dragHideEdges {Function}

拖动边隐藏行为

wheelZoomHideEdges {Function}

滚轮缩放边隐藏行为

shortcut {Function}

快捷键行为。网图（Net）专有

collapse {Function}

树图的折叠行为。树图（Tree）专有

spreadout {Function}

树图的展开行为。树图（Tree）专有
