Item

Item 子项，是 G6 内部节点 (node)和边边 (edge)的抽象概念。本文档将向大家说明，节点和边已开放的 API 。

  • 节点——Node
  • 边——Edge

节点——Node

方法

显示

hide {Function}

隐藏

获取关键形

getBBox {Function}

获取包围盒

getModel {Function}

获取数据模型

getGroup {Function}

获取图形图组

getShapeCfg {Function}

获取绘制配置项

getAnchor {Function}

获取锚点

/**
 * 获取锚点
 * @param  {Object} 输入是一个点则返回该点对应的锚点 {x: number, y: number}
 * @return {Object} point 锚点 {x: number, y: number}
 */
node.getAnchor(param);
