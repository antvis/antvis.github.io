G2

全局 G2 对象，在 script 标签引入 g2.js 文件后获得，或者在 AMD 环境中通过 require('g2') 获得。

通过 G2 对象可调用的属性：

  • G2.Chart
  • G2.Util
  • G2.Global
  • G2.Stat
  • G2.Scale
  • G2.Shape
  • G2.Frame
  • G2.Layout
  • G2.Animate
  • G2.Canvas
  • G2.Coord

温馨提示： 由于内部业务需要，所以我们在 G2 的代码中加上了版本使用情况的打点监控，如果您不想让我们知道您的版本使用情况，请使用以下方式避免：

1、 对于 <script> 标签的引用方式，如果您使用的是 2.1.4 之前的版本，请使用以下链接(使用时请将 version 替换为具体的版本号)：

<script src="https://npmcdn.com/g2@{version}/index.js"></script>

2、g2@2.1.4 版本开始，我们添加了打点监控开关 G2.track(boolean)，默认打开，通过设置 G2.track(false) 即可关闭。
